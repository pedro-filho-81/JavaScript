Eventos JavaScript

Pode-se dizer que eventos são ações capazes de disparar uma reação.

Exemplo do evento onclick do elemento button do Html.

Clique no botão para chamar os métodos alert(), confirm() e prompt().

Usar eventos como atributos de um elemento HTML, está se misturando camadas de comportamento com a camada de estruturação e isso não é uma prática em conformidade com os Padrões Web.

      <p><button type="button" 
      onclick="alert('Eu sou o método alert()')">
      Alert</button>

      <button type="button" 
      onclick="confirm('Eu sou o método confirm, 
      está gostando?')">Confirm</button>
   
      b<utton type="button" 
      onclick="prompt('Qual seu nome?', 
      'digite seu nome aqui.')">Prompt</button></p>
