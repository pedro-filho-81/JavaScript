Números Vizinhos

Escreva um programa que receba um número e exiba o anterior e o posterior.

   
   <script>
      // variáveis
      const n = Number(prompt('Digite um número inteiro: ') );
      
      // exibe o resultado
      alert(`Você digitou o número ${n}
      \no anterior é ${(n - 1)}
      \no posterior é ${n + 1}`);
   </script>
