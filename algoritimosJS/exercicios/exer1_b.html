Valor por Cliente

Programa para calcular o pagamento da conta e mostrar o valor por clientes. 

 
   <script>
      // variáveis
      const conta = Number(prompt('Informe o valor da conta R$ '));
      
      const numClientes = Number(prompt('Informe o número de clientes: ') );

      const totPorCli = conta / numClientes;

      // exibe resultado
      alert(`Valor da conta R$ ${conta.toFixed(2).
         replace('.', ',')}
      \nNúmero de clientes ${numClientes}
      \nTotal por cliente R$ ${totPorCli.toFixed(2).
         replace('.',',')}`);
   </script>
