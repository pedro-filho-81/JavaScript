Pagamento com Desconto

c - Elaborar um programa para uma loja, o qual leia o preço de um produto e informe as opções de pagamento da loja. Calcule e informe o valor para pagamento à vista com 10% de desconto e o valor em 3x. Exemplo: Preço R$: 60.00 À Vista R$: 54.00 Ou 3x de R$: 20.00
Iepsen, Edécio Fernando. Lógica de Programação e Algoritmos com JavaScript - 2ª Edição (p. 51). Novatec Editora. Edição do Kindle. 

   <script>
   // variáveis
   // preço recebe o valor do usuário
   const preco = Number(prompt('Informe o preço do produto R$ '));

   // cálculos
   const desconto = preco * 0.10; // 10 % de descontos
   const avista = preco - desconto; // valor com descontos
   const prestacao = preco / 3; // pagamento a prestação

   // exibe resultado
   alert(`Preço do produto R$ 
   ${preco.toFixed(2).replace('.', ',')}
   
   \nDesconto R$ ${desconto.toFixed(2).replace('.', ',')}
   
   \nÀ vista R$ ${avista.toFixed(2).replace('.', ',')}
   
   \nOu 3x de R$ ${prestacao.toFixed(2).replace('.', ',')}`); 
   </script>
