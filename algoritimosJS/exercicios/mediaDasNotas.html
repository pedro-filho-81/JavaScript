Média das Notas

Elaborar um programa que leia 2 notas e calcule a média delas.

   <script>
   // variáveis
   const nota1 = Number(prompt('Informe a primeira nota: '));
   const nota2 = Number(prompt('Informe a segunda nota: '));

   // calcula média
   const media = (nota1 + nota2) / 2;

   // exibe resultado
   alert(`A média das notas ${nota1} e ${nota2} é ${media}`);
   </script>
