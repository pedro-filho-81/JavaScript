Document Write

Este método escreve diretamente na página HTML, usando tags HTML.

OBS: De acordo com os padrões WEB, este método deve ser evidado.

O método writeln() é semelhante ao método write(), com a diferença que acrescenta um caractere de nova linha ao final do conteúdo inserido.

   <script>
   document.write("<h2>O método <code>write</code></h2>>");

   document.write("<p> método <code> write</code> do objeto 
   <code>document</code> destina-se a escrever marcação HTML 
   com uso da JavaScript.</p>>");
   </script>
