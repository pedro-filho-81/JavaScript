Calcula Valor do Jantar

Programa para calcular o valor de um jantar.

const jantar = Number(prompt("Informe o valor do jantar: R$ ")); // lê o valor do jantar
const garcon = jantar * 0.10; //c10% do garçom
const total = jantar + garcon; // calcula o total da conta
// exibe o valor da conta
alert(`Valor a pagar é de:\njantarR$ ${jantar.toFixed(2).replace('.', ',')}
\ngarçon R$ ${garcon.toFixed(2).replace('.', ',')}
\nTotal R$ ${total.toFixed(2).replace('.', ',')}`);
