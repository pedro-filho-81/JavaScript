Calcula o Dobro de um Número

Calcula o Dobro de um número usando o sinal de mais ( + )

Se for calcular o dobro do número a com o operador aritimético ( + ) adição,
para a conversão é necessário utilizar o método Number() direto na entrada de dados.

const n = Number( prompt("Digite um número qualquer: ") );
const dobro = n + n;
alert(`O dobro de ${n} é ${dobro}`);
