JavaScript no Html:5

A tag <script>

Em HTML, o código JavaScript é inserido entre e tags.<script</script>

   Exemplo
   <script>
      document.getElementById("demo").innerHTML = "Meu Primeiro JavaScript";
   </script>

Exemplos antigos de JavaScript podem usar um atributo type: <script type="text/javascript">.

O atributo type não é obrigatório. JavaScript é a linguagem de script padrão em HTML.

Funções e eventos JavaScript

function é um bloco de código JavaScript, que pode ser executado quando "solicitado".

Por exemplo, uma função pode ser chamada quando o usuário clica em um botão.

JavaScript em <head> ou <body>

Você pode colocar qualquer número de scripts em um documento HTML.

Os scripts podem ser colocados no <head>, ou no <body> de uma página HTML, ou em ambos.

JavaScript em <head>

Neste exemplo, um JavaScript é colocado na seção <head> de uma página HTML.

A função é invocada (chamada) quando um botão é clicado:

      
   Exemplo
   <!DOCTYPE html>
   <html>
   <head>
   <script>
      function myFunction() {
         document.getElementById("demo").innerHTML = "Alteração de Parágrafo.";
      }
   </script>
   </head>
   <body>

   Demo JavaScript na cabeça

   <p id="demo">Um Parágrafo</p>
   <button type="button" onclick="myFunction()">Experimente</button>

   </body>
   </html>

JavaScript em <body>

Neste exemplo, um JavaScript é colocado na seção de uma página HTML.function() na <body>

A função é invocada (chamada) quando um botão é clicado:

   Exemplo
   <!DOCTYPE html>
   <html>
   <body>

   <h2>Demo JavaScript no Corpo</h2>

   <p id="demo">Um Parágrafo</p> 

   <button type="button" onclick="myFunction()">Esperimente</button>

   <script>
      function myFunction() {
         document.getElementById("demo").innerHTML = "Alteração do Parágrafo.";
      }
   </script>

   </body>
   </html>

Colocar scripts na parte inferior do elemento <body> melhora a
exibição velocidade, porque a interpretação de script deixa a exibição mais lenta.

JavaScript externo

Os scripts também podem ser colocados em arquivos externos:

   Arquivo externo: myScript.js
      function myFunction() {
         document.getElementById("demo").innerHTML = "Alteração do Parágrafo.";
      }

Scripts externos são práticos quando o mesmo código é usado em muitas páginas da Web diferentes.

Os arquivos JavaScript têm a extensão de arquivo .js.

Para usar um script externo, coloque o nome do arquivo de script no atributo (source) de uma tag:src<script>

   Exemplo
   <script src="myScript.js"></script>

Você pode colocar uma referência de script externo em <head> ou no <body> como desejar.

O script se comportará como se estivesse localizado exatamente onde a tag <script> está localizada.

Scripts externos não podem conter tags.<script>

Vantagens do JavaScript externo

Colocar scripts em arquivos externos tem algumas vantagens:

      Ele separa HTML e código
      Torna HTML e JavaScript mais fáceis de ler e manter
      Arquivos JavaScript armazenados em cache podem acelerar 
      o carregamento de páginas.

Para adicionar vários arquivos de script a uma página
- use várias marcas de script: 

      Exemplo
      <script src="myScript1.js"><script>
      <script src="myScript2.js"></script>

Referências Externas

Um script externo pode ser referenciado de 3 maneiras diferentes:

         
         Exemplo 1
         Este usa uma URL completa para vincular a myScript.js:      
         <script src="https://www.w3schools.com/js/myScript.js"><script>

         Exemplo 2
         Este usa um caminho de arquivo para vincular a myScript.js:
         <script src="/js/myScript.js">

         Exemplo 3
         Este não usa nenhum caminho para vincular a myScript.js:
         <script src="myScript.js"></script>
