Sintax JavaScript

A sintaxe JavaScript é o conjunto de regras, como os programas JavaScript são construídos:

      // Como criar variáveis
      // How to create variables:
      var x;
      let y;
   
      // Como usar variáveis
      // How to use variables:
      x = 5;   // x recebe 5
      y = 6;   // y recebe 6
      let z = x + y; // z recebe a soma de x e y

Valores

A sintaxe JavaScript define dois tipos de valores:

      valores fixos
      valores variáveis

Valores fixos são chamados de Literais.

Os valores das variáveis são chamados de Variáveis.

Literais

As duas regras de sintaxe mais importantes para valores fixos são:

   1. Os números são escritos com ou sem casas decimais:
      123456
      10.50
      1001
      1,225.14
   
   2. Strings são textos, escritos entre aspas duplas ou simples:
      "Este é um exemplo de string."
      "Pedro Filho"
      'Cleonice'

Variáveis

Em uma linguagem de programação, as variáveis são usadas para armazenar valores de dados.

JavaScript usa as palavras-chave var, let e const para declarar váriaveis.

Um sinal de igual é usado para atribuir valores a variáveis.

Neste exemplo, x é definido como uma variável. Então, ao x é atribuído (dado) o valor 6:

      let x; // cria variável
      x = 6; // x recebe 6

Operadores

JavaScript usa operadores aritméticos ( + - * /) para calcular valores:

      (5 + 6) * 10
      1 + 2 - 5

JavaScript usa um operador de atribuição ( = ) para atribuir valores a variáveis: 

   let x, y; // cria variáveis
   x = 5;   // x recebe 5
   y = 6;   // y recebe 6

Expressões

Uma expressão é uma combinação de valores, variáveis operadores, que calcula um valor.

O cálculo é chamado de avaliação.

Por exemplo, 5 * 10 resulta em 50: 5 * 10

As expressões também podem conter valores variáveis: x * 10

Os valores podem ser de vários tipos, como números e strings.

Por exemplo, "John" + " " + "Doe", resulta em "John Doe".

Palavras-chave

Palavras-chave JavaScript são usadas para identificar ações a serem executadas.

A palavra-chave let diz ao navegador para criar variáveis:

      let x, y;
      x = 5 + 6;
      y = x * 10;

A palavra-chave var também diz ao navegador para criar variáveis:

      var x, y;
      x = 5 + 6;
      y = x * 10;

Nesses exemplos, usar var ou let produzirá o mesmo resultado.

Comentários

Nem todas as instruções JavaScript são "executadas".

Código após barras duplas // ou entre /* e */ é tratado como um comentário .

Os comentários são ignorados e não serão executados:

      let x = 5;   // esta mensagem não será exibida
      // x = 6;  esta instrução não será exibida

Identificadores / Nomes

Identificadores são nomes em JavaScript.

Os identificadores são usados para nomear variáveis, palavras-chave e funções.

As regras para nomes (legais) são as mesmas na maioria das linguagens de programação.

Um nome deve começar com:

      Uma letra (AZ ou az)
      Um cifrão ($)
      Ou um sublinhado (_)
      Os caracteres subseqüentes podem ser letras, 
         dígitos, sublinhados ou cifrões.

Números não são permitidos como primeiro caractere em nomes.

Dessa forma, o JavaScript pode facilmente distinguir identificadores de números.

JavaScript diferencia maiúsculas de minúsculas

Todos os identificadores JavaScript diferenciam maiúsculas de minúsculas.

As variáveis lastName e lastname, são duas variáveis diferentes:

      let lastname, lastName; // cria variáveis
      lastName = "Doe";
      lastname = "Peterson";

JavaScript não interpreta LET ou Let como a palavra-chave let.

JavaScript e Camel Case

Historicamente, os programadores usaram diferentes maneiras de unir
várias palavras em um nome de variável:

Hífens não são permitidos em JavaScript. Eles são reservados para subtrações.

   Hífens:
      nome, 
      sobre-nome, 
      cartão-master, 
      inter-municipal.
   
   Sublinhado:
      primeiro_name, 
      ultimo_name, 
      master_card, 
      inter_city.
   
   Upper Camel Case (Pascal Case):
      Nome, 
      Sobrenome, 
      Mastercard, 
      Intercity.
   
   CamelCase:
      Nome, 
      SobreNome, 
      MasterCard, 
      InterCity.
   
   Caixa Camelo Inferior:
      Os programadores de JavaScript tendem a usar maiúsculas e minúsculas 
      que começam com uma letra minúscula:
      firstName, 
      lastName, 
      masterCard, 
      interCity.

Conjunto de caracteres

JavaScript usa o conjunto de caracteres Unicode.

O Unicode cobre (quase) todos os caracteres, pontuações e símbolos do mundo.
