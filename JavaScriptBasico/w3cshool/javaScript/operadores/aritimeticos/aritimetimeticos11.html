Operadores Aritiméticos JavaScript

São usados para executar operações aritméticas nos operandos. Os operadores a seguir são conhecidos como operadores aritméticos JavaScript.

   Operador	Descrição	   Exemplo
   +	       Adição              10+20 = 30
   -	       Subtração	   20-10 = 10
   *	       Multiplicação	   10*20 = 200
   /	       Divisão	           20/10 = 2
   %	       Módulo (Resto)      20%10 = 0
   **          Exponenciação       8 **4 = 4096
   ++	       Incremento	   var a=10; a++; Agora a = 11
   --	       Decréscimo	   var a=10; um--; Agora a = 9

Operações aritméticas

Uma operação aritmética típica opera em dois números.

Os dois números podem ser:

   Literais:
   Exemplo
   let x = 100 + 50;

   ou variáveis:
   Exemplo
   let x = a + b;
   
   ou expressões:
   Exemplo
   let x = (100 + 50) * a;

Operadores e Operandos

Os números (em uma operação aritmética) são chamados de operandos.

A operação (a ser executada entre os dois operandos) é definida por um operador.

O operador de adição ( + ) adiciona números:

   Exemplo
   let x = 5;
   let y = 2;
   let z = x + y;

Subtraindo

O operador de subtração ( - ) subtrai números.

   Exemplo
   let x = 5;
   let y = 2;
   let z = x - y;

Multiplicando

O operador de multiplicação ( * ) multiplica números.

   Exemplo
   let x = 5;
   let y = 2;
   let z = x * y;

Dividindo

O operador de divisão ( / ) divide números.

   Exemplo
   let x = 5;
   let y = 2;
   let z = x / y;

Restante

O operador de módulo ( % ) retorna o restante da divisão.

   Exemplo
   let x = 5;
   let y = 2;
   let z = x % y;

Em aritmética, a divisão de dois inteiros produz um quociente e um resto.

Na matemática, o resultado de uma operação modulo é o restante de uma divisão aritmética.

Incrementar

O operador de incremento ( ++ ) incrementa números.

   Exemplo
   let x = 5;
   x++;
   let z = x;

Decrementar

O operador de decréscimo ( -- ) decreta números.

   Exemplo
   let x = 5;
   x--;
   let z = x;

Exponenciação

O operador de exponenciação ( ** ) eleva o primeiro operando à potência do segundo operando.

   Exemplo
   let x = 5;
   let z = x ** 2;
   
   x ** y produz o mesmo resultado que : Math.pow(x,y)

   Exemplo
   let x = 5;
   let z = Math.pow(x,2);

Precedência do operador

A precedência do operador descreve a ordem em que as operações são executadas em uma expressão aritmética.

   Exemplo
   let x = 100 + 50 * 3;

É o resultado do exemplo acima do mesmo que 150 * 3, ou é o mesmo que 100 + 150?

A adição ou a multiplicação é feita primeiro?

Como na matemática escolar tradicional, a multiplicação é feita primeiro.

Multiplicação ( * ) e divisão ( / ) têm precedência maior do que adição ( + ) e subtração ( - ).

E (como na matemática escolar) a precedência pode ser alterada usando parênteses.

Ao usar parênteses, as operações dentro dos parênteses são computadas primeiro:

   Exemplo
   let x = (100 + 50) * 3;

Quando muitas operações têm a mesma precedência (como adição e subtração ou multiplicação e divisão), eles são calculados da esquerda para a esquerda Direita:

   Exemplos
   let x = 100 + 50 - 3;
   let x = 100 / 50 * 3;
