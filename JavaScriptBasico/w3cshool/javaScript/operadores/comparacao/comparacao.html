Operadores de Comparação

O operador de comparação compara os dois operandos.

Os operadores de comparação são os seguintes:

   Operador	         Descrição   	      Exemplo
   ==	                        É igual a	         10==20 = falso
   ===	            Idêntico (igual e do mesmo tipo)	10==20 = falso
   !=	               Não é igual a	      10!=20 = verdadeiro
   !==	            Não idêntico	      20!==20 = falso
   >	               Maior que	         20>10 = verdadeiro
   >=	               Maior ou igual a	   20>=10 = verdadeiro
   <	            Menos de	            20 < 10 = falso
   <=	            Menor ou igual a	   20<=10 = falso

O operador é igual a ==

O operador == compara se um valor é igual ao outro:

Atribua 5 a variável x, exiba true ou false com base
nos valores e no tipo da variável da comparação:

let x = 5; cria uma variável do tipo inteiro.

      <body> // aqui html
      <!--Os parágrafos com o (id="demo") vão ser alterados pelo JavaScript-->
      <p id="demo"></p>
      <p id="demo2"></p>
      <p id="demo3"></p>

      <script> // aqui javascript dentro do HTml
      
         let x = 5; // declara variável do tipo inteiro

         // aqui a comparação é entre valores diferentes e do mesmo tipo.
         document.getElementById("demo").innerHTML = ("x é igual a 8 ?" + (x == 8);

         // aqui a comparação é entre valores iguais e do mesmo tipo.
         document.getElementById("demo2").innerHTML = ("x é igual a 5? " + (x == 5);

         // aqui a comparação é entre um inteiro e uma string
         // o operador == compara valores e não o tipo da variável.
         // o valor 5 inteiros é igual a 5 string resultado verdadeiro, são iguais.
         document.getElementById("demo3").innerHTML = ("x é igual a string 5? " + (x == "5");

         </script>
      </body>
      Saída:
            x é igual a 8: false
            x é igual a 5: true
            x é igual a string 5: true

O operador é idêntico a ===

O operador === compara os valores para saber se são idênticos,
ou seja, valor1 é idêntico ao valor2? (valor e tipo):
Se sim retorna true, se não retorna false.

Atribua 5 a variável x, exiba true ou false com base
nos valores e no tipo da variável da comparação:

let x = 5; cria uma variável do tipo inteiro.

      <body> // aqui html
         <!--Os parágrafos com o (id="demo") vão ser alterados pelo JavaScript-->
         <p id="demo"></p>
         <p id="demo2"></p>
         <p id="demo3"></p>

         <script> // aqui javascript dentro do HTml
            let x = 5; // declara variável do tipo inteiro

            // aqui a comparação é entre dois valores diferentes e do mesmo tipo.
            document.getElementById("demo").innerHTML = ("x é idêntico a 8 ?" + (x === 8);

            // aqui a comparação é entre dois valores iguais e do mesmo tipo.
            document.getElementById("demo2").innerHTML = ("x é idêntico a 5? " + (x === 5);

            // aqui a comparação é entre um valor inteiro e uma string
            // o operador === compara o valor e  tipo da variável.
            // o valor 5 inteiros é idêntico a 5 string resultado é false, Não são idênticos.
            document.getElementById("demo3").innerHTML = ("x é idêntico a string 5? " + (x === "5");

         </script>
         </body>
            Saída:
               x é idêntico a 8: false
               x é idêntico a 5: true
               x é idêntico a string 5: false

O operador NÃO é idêntico a !==

O operador !== compara os valores para saber se NÃO são idênticos a,
ou seja, valor1 NÃO é idêntico ao valor2? (valor e tipo):
Se sim retorna true, se não retorna false.

Atribua 5 a variável x, exiba true ou false com base
nos valores e no tipo da variável da comparação:

let x = 5; cria uma variável do tipo inteiro.

      <body> // aqui html
         <!--Os parágrafos com o (id="demo") vão ser alterados pelo JavaScript-->

         <p id="demo2"></p>
         <p id="demo3"></p>

         <script> // aqui javascript dentro do HTml
            let x = 5; // declara variável do tipo inteiro

            // aqui a comparação é entre dois valores diferentes e do mesmo tipo.
            // 5 NÃO é idêntico a 8
            document.getElementById("demo").innerHTML = ("x NÃO é idêntico a 8 ?" + (x !== 8); 
            // resultado verdadeiro

            // aqui a comparação é entre dois valores iguais e do mesmo tipo.
            // 5 NÃO é idêntico a 5
            document.getElementById("demo2").innerHTML = ("x NÃO é idêntico a 5? " + (x !== 5); 
            // resultado falso

            // aqui a comparação é entre um valor inteiro e uma string
            // o operador !== compara o valor e  tipo da variável.
            // o valor 5 inteiros NÃO é idêntico a string 5.

               x NÃO é idêntico a 8: true
               x NÃO é idêntico a 5: false
               x NÃO é idêntico a string 5: true
