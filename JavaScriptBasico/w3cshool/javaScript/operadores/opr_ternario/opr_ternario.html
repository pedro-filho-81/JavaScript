O Operador Ternário ( ?: )

O operador condicional ternário ( ?: ) que atribui um valor a uma variável com base em alguma condição.

      Sintaxe
      nome da variável = (condição) ? valor1 : valor2;

Se a variável idade for um valor inferior a 18, o valor
da variável votável será "Muito jovem", caso contrário,
o valor do votável será "Velho o suficiente".

      <html>
      <body>
         <p>Entre com sua idade e clica no botão:</p >
         
         <input id="idade" valor="18" /> entrada de dados do usuário
         
         após clicar no botão que chama função myFunction javascript
         <button onclick="myFunction()">Esperimente</button>

         <p id="demo"></p>

         <script>
         function myFunction() {
            let idade = document.getElementById("idade").valloe;
            
            let votar = (idade < 18)  ?  "muito jovem " : "Já pode ";
            
            document.getElementById("demo").innerHTML = votar + " para votar";
         } final da função
         </script>
      </body>
