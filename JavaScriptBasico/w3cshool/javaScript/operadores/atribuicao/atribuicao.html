Operadores de Atribuição

Os operadores a seguir são conhecidos como operadores de atribuição JavaScript.

   Operador	   Descrição	            Exemplos
   =                      Atribuir	               10+10 = 20
   +=	                  Adicionar e atribuir	var a=10; a+=20; Agora a = 30
   -=	                  Subtrair e atribuir	var a=20; a-=10; Agora a = 10
   *=	                  Multiplicar e atribuir	var a=10; a*=20; Agora a = 200
   /=	                  Dividir e atribuir	var a=10; a/=2; Agora a = 5
   %=	          Módulo e atribuição	var a=10; a%=2; Agora a = 0
   **=                  Expoente e atribuição  var a=10; a**=2 Agora a = 100

O Operador =

O Operador de Atribuição Simples atribui um valor a uma variável.

   Exemplos de atribuição simples
   let x = 10;
   let x = 10 + y;

O operador +=

O Operador de Atribuição de Adição adiciona um valor a uma variável.

   Exemplos de atribuição de adição
   Número:
   let x = 10;
   x += 5;
   
   String:
   let text = "Hello"; 
   text += " World";
   
   Saída:
   Hello World
   
   O sinal de ( + ) com Strings é o sinal de concatenação.
   Concatenação é a junção de duas ou mais (palavras, frases, etc...);

O operador -=

O Operador de Atribuição de Subtração subtrai um valor de uma variável.

   Exemplo de atribuição de subtração
   let x = 10;
   x -= 5;

O Operador *=

O Operador de Atribuição de Multiplicação multiplica uma variável.

   Exemplo de atribuição de multiplicação
   let x = 10;
   x *= 5;

O Operador **=

O Operador de Atribuição de Exponenciação eleva uma variável à potência do operando.

   Exemplo de atribuição de exponenciação
   let x = 10;
   x **= 5;

O operador /=

O Operador de Atribuição de Divisão divide uma variável.

   Exemplo de atribuição de divisão
   let x = 10;
   x /= 5;

O operador %=

O Operador de Atribuição de Restante atribui um restante a uma variável.

   Exemplo de atribuição restante
   let x = 10;
   x %= 5;
