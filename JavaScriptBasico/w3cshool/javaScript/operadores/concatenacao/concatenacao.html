Operador de Concatenação ( + )

O operador ( + ) também pode ser usado para adicionar (concatenar) cadeias de caracteres.

      <body>
      <p>O perador ( + ) concatena (adiciona, une) cadeias de caracteres.</p>
      <p id="demo"></p> // (id demo) será substituido pelo texto3 do javascript

      <script>
      let texto1 = "Pedro";
      let texto2 = "Filho";
      let texto3 = texto1 + " " + texto2;
      document.getElementById("demo").innerHTML = texto3 ;
      </script>
      </body>

      Saída:
         O operador + concatena (adiciona, une) cadeia de caracteres.
         Pedro Filho <= é resultado da concatenação de texto1 + " " + texto2.

O operador de atribuição ( += ) também pode ser usado para adicionar (concatenar) cadeias de caracteres:

      <body>
      <p>O perador de atribuição ( += ) pode concatenar cadeias de caracteres.</p>
      <p id="demo"></p> // (id demo) será substituido pelo texto1 do javascript

      <script>
      var texto1 = "Estou fazendo esse ";
      texto1 += "exercício no sábado.";
      document.getElementById("demo").innerHTML = texto1;
      </script>
      </body>

      Saída:
         O perador de atribuição ( += ) pode concatenar cadeias de caracteres.
         Estou fazendo esse exercício no sábado.

Quando usado em cadeias de caracteres, o operador ( + ) é chamado de operador de concatenação.

Se você adicionar um número e uma cadeia de caracteres, o resultado será uma cadeia de caracteres!

      <body>
      <p>O perador aritimético ( + ) concatena (adiciona, une) cadeias de caracteres.</p>
      <p id="demo"></p> // (id demo) será substituido pelo texto3 do javascript

      <script>
      let texto1 = "Pedro";
      let texto2 = 123456;
      let texto3 = texto1 + " " + texto2;
      document.getElementById("demo").innerHTML = texto3 ;
      </script>
      </body>

      Saída:
         O operador aritimético + concatena (adiciona, une) cadeia de caracteres.
         Pedro 123456 <= é resultado da concatenação de texto1 + " " + texto2.
