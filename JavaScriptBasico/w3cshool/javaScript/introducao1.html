Introdução ao JavaScript

Esta página contém alguns exemplos do que o JavaScript pode fazer.

Por que estudar JavaScript?

JavaScript é uma das 3 linguagens que todos os desenvolvedores web devem aprender:

   1. HTML para definir o conteúdo das páginas web
   
   2. CSS para especificar o layout das páginas da web
   
   3. JavaScript para programar o comportamento de páginas da web

JavaScript pode alterar o conteúdo HTML

Um dos muitos métodos HTML JavaScript é .getElementById()

A instrução abaixo "localiza" um elemento HTML com o id="demo",
e altera o conteúdo do elemento innerHTML para "Olá, JavaScript!":

   Exemplo
   <!DOCTYPE html>
   <html>
   <body>
      <h2>O que JavaScript Pode Fazer?</h2>

      <p id="demo">JavaScript pode mudar o conteúdo HTML.</p>

      // cria um botão que quando clicado 
      // faz a alteração do parágrafo que tem
      // o ID="demo".
      <button type="button" onclick='document.getElementById("demo").
      innerHTML = "Olá, JavaScript!"'>
      Click Me!</button>
   </body>
   </html>

JavaScript aceita aspas duplas e simples:

Neste exemplo, as duas instruções são válidas.

   Exemplo
   document.getElementById('demo').innerHTML = 'Hello JavaScript';
   
   document.getElementById("demo").innerHTML = "Hello JavaScript";
   /*
      A instrução acima diz: no documento obtenha do elemento 
      que está identificado pelo Id demo no Html e atribua a ele  
      o que está entre aspas (duplas ou simplas).
   */

JavaScript pode alterar valores de atributo HTML

Neste exemplo, o JavaScript altera o valor do atributo source de uma tag: src <img>

   Exemplo
   <img id="myImage" src="pic_bulboff.gif" style="width:100px">

JavaScript pode alterar estilos HTML (CSS)

Alterar o estilo de um elemento HTML é uma variante da alteração de um atributo:

   <body>
      <p id="demo">JavaScript pode mudar o estilo de um elemento HTML.</p>
      
      // cria um botão que quando clicado 
      // faz a alteração do parágrafo que tem
      // o ID="demo".
      <button type="button" onclick="document.getElementById('demo').style.fontSize='35px'">
      Click Me!</button>
   </body>
      Saída (com a frase no tamanho 35px)
      JavaScript pode mudar o estilo de um elemento HTML

JavaScript pode ocultar e mostrar elementos HTML

Para ocultar ou mostrar o elemento HTML podemos alterando o estilo: display

      <!DOCTYPE html>
      <html>
      <body>
         <p id="demo">JavaScript pode ocultar elementos HTML.</p>
         
         instrução para ocultar
         document.getElementById("demo").style.display = "none";

         // cria um botão que quando clicado 
         // faz a alteração do parágrafo que tem
         // o ID="demo".
         <button type="button" onclick="document.getElementById('demo').style.display='none'">
         Click Me!</button>
      </body>
      </html>

JavaScript pode mostrar elementos HTML

Mostrar elementos HTML ocultos também pode ser feito alterando o estilo: display

      <!DOCTYPE html>
      <html>
      <body>

      <p id="demo" style="display:none">Hello JavaScript!</p>

      instrução para mostrar
      document.getElementById("demo").style.display = "block";

      // cria um botão que quando clicado 
      // faz a alteração do parágrafo que tem
      // o ID="demo".
      <button type="button" onclick="document.getElementById('demo').style.display='block'">
      Click Me!</button>

      </body>
      </html>

Curuisidades

JavaScript e Java são linguagens completamente diferentes, ambas em conceito e design.

JavaScript foi inventado por Brendan Eich em 1995, e tornou-se um padrão ECMA em 1997.

ECMA-262 é o nome oficial da norma. ECMAScript é o nome oficial da língua.
