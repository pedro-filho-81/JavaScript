A Palavra-Chave Let

A palavra-chave let foi introduzida no ES6 (2015).

As variáveis definidas com let não podem ser Redeclared.

As variáveis definidas com let devem ser declaradas antes do uso.

As variáveis definidas com let têm Escopo de Bloco.

Não pode ser redeclarado

As variáveis definidas com let não podem ser redeclaradas.

Você não pode redeclarar acidentalmente uma variável.

Com let você não pode fazer isso:

   Exemplo
   let x = "John Doe"; // variável recebe string
      
   let x = 0;          // variável x redeclarada recebe 0 
      
      // SyntaxError: 'x' has already been declared

Você pode fazer isso com var

   Exemplo
   var x = "John Doe";
      
   var x = 0;

Escopo do bloco

Antes do ES6 (2015), o JavaScript tinha apenas o Escopo Global e o Escopo da Função.

O ES6 introduziu duas novas palavras-chave JavaScript importantes: e .let const

Essas duas palavras-chave fornecem Block Scope em JavaScript.

As variáveis declaradas dentro de um bloco { } não podem ser acessadas do exterior do bloco:

   Exemplo
      {
         let x = 2;
      }
      // x não pode ser usado aqui

As variáveis declaradas com a palavra-chave var NÃO podem ter escopo de bloco.var

As variáveis declaradas dentro de um bloco { } podem ser acessadas a fora do bloco.

   Exemplo
      {
         var x = 2;
      }
      // x pode ser usada aqui

Redeclarando Variáveis

Redeclarar uma variável usando a palavra-chave var pode impor problemas.

Redeclarar uma variável dentro de um bloco também redeclarará a variável fora do bloco:

   Exemplo
      var x = 10; // aqui x é 10
      {
         var x = 2; // aqui x é 2
      }
      // aqui x é 2

Redeclarar uma variável usando a palavra-chave let pode resolver esse problema.

Redeclarar uma variável dentro de um bloco não redeclarará a variável fora o bloco:

   Exemplo
      let x = 10; // aqui x é 10
      {
         let x = 2; // Here x is 2
      }
      // aqui x é 10

Redeclarando

A declaração novamente de uma variável JavaScript com var é permitida em qualquer lugar em um programa:

   Exemplo
      var x = 2; // agora x é 2

      var x = 3; // agora x é 3

Com let, NÃO é permitido redeclarar uma variável no mesmo bloco:

   Exemplo
      var x = 2;   // permitido
      let x = 3;   // nâo permitido

      {
         let x = 2;   // permitido
         let x = 3;   // nâo permitido
      }

      {
         let x = 2;   // permitido
         var x = 3;   // nâo permitido
      }

Redeclarar uma variável com let, em outro bloco, é permitido:

   Exemplo
      let x = 2;   // permitido

      {
         let x = 3;   // permitido
      }

      {
         let x = 4;    // permitido
      }
