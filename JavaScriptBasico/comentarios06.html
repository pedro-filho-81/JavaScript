Comentários JavaScript

Os comentários JavaScript podem ser usados para explicar o código JavaScript e torná-lo mais legível.

Os comentários do JavaScript também podem ser usados para impedir a execução, ao testar o código alternativo.

Comentários de linha única

Os comentários de linha única começam com //.

Qualquer texto entre // e no final da linha será ignorado pelo JavaScript (não será executado).

Este exemplo usa um comentário de uma única linha antes de cada linha de código:

   Exemplo
      // Change heading:
      document.getElementById("myH").innerHTML = "My First Page";
   
      // Change paragraph:
      document.getElementById("myH").innerHTML = "My First Page";

Este exemplo usa um comentário de linha única no final de cada linha para explicar o código:

   Exemplo
      let x = 5;      // Declare x, give it the value of 5
      let y = x + 2;  // Declare y, give it the value of x + 2

Comentários de várias linhas

Os comentários de várias linhas começam com /* e terminam com */.

Qualquer texto entre /* e */ á ignorado pelo JavaScript.

Este exemplo usa um comentário de várias linhas (um bloco de comentários) para explicar o código:

   Exemplo
      /*
         The code below will change
         the heading with id = "myH"
         and the paragraph with id = "myP"
         in my web page:
      */
      document.getElementById("myH").innerHTML = "My First Page";
      document.getElementById("myP").innerHTML = "My First paragraph";

É mais comum usar comentários de linha única.

Os comentários em bloco costumam ser usados para documentação formal.

Usando comentários para impedir a execução

O uso de comentários para impedir a execução do código é adequado para teste de código.

Adicionar // a frente de uma linha de código altera as
linhas de código de uma linha executável para um comentário.

Este exemplo usa // para impedir a execução de uma das linhas de código:

   Exemplo
      // document.getElementById("myH").innerHTML = "My First Page";
      document.getElementById("myH").innerHTML = "My First paragraph";

Este exemplo usa um bloco de comentário para impedir a execução de várias linhas:

   Exemplo
      /*
         document.getElementById("myH").innerHTML = "My First Page";
         document.getElementById("myH").innerHTML = "My First paragraph";
      */
